Faça o push do projeto para o GitHub seguindo estes passos:

1. Rode `git status` para ver o que mudou
2. Rode `git add -A`
3. Faça o commit com a mensagem: "$ARGUMENTS"
   - Se nenhuma mensagem foi fornecida, crie uma mensagem descritiva baseada nos arquivos modificados
4. Rode `git push`

Informe ao usuário quais arquivos foram incluídos e confirme que o push foi realizado com sucesso.
