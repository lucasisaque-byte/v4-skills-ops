# Template — Rebriefing (Conversão de Feedback)

## Rebriefing parcial (ajuste pontual)

Use quando o usuário pediu mudanças específicas sem rejeitar a peça toda.

```
## REBRIEFING PARCIAL — [CLIENTE] — [DATA]

**Peça original:** [referência]
**Feedback recebido:** "[feedback verbatim do usuário]"

---

### O que manter
[Tudo que não foi mencionado no feedback — liste explicitamente para não perder]

### O que ajustar
| Elemento | Situação atual | Instrução de ajuste |
|----------|---------------|---------------------|
| [headline] | [texto atual] | [o que mudar] |
| [CTA] | [texto atual] | [o que mudar] |
| [tom] | [como está] | [como deve ficar] |

### Instrução para a skill
[Instrução direta e clara para a skill revisar apenas os pontos indicados,
mantendo o restante intacto]
```

---

## Rebriefing completo (refação total)

Use quando o usuário rejeitou o ângulo, mudou de direção ou ficou insatisfeito com a peça como um todo.

```
## REBRIEFING COMPLETO — [CLIENTE] — [DATA]

**Peça anterior:** [referência — não será aproveitada]
**Motivo da refação:** [o que levou à decisão de recomeçar]
**Nova direção do usuário:** "[instrução verbatim do usuário]"

---

### O que aprendemos com a tentativa anterior
[O que não funcionou e por quê — para não repetir]

### Novo briefing
[Briefing completo com a nova direção incorporada]
[Usar o template padrão da skill correspondente]

### Alertas
[Riscos ou lacunas que persistem mesmo com a nova direção]
```

---

## Como diferenciar ajuste de refação

**Sinais de ajuste pontual:**
- Menciona elementos específicos ("muda só o título")
- Tom construtivo ("gostei mas...")
- Escopo limitado ("troca o CTA")
- Mantém a aprovação do ângulo geral

**Sinais de refação total:**
- Rejeita o ângulo ou a premissa ("esse caminho não é o que eu quero")
- Muda a persona ou o objetivo ("na verdade é para o CFO, não para o Head de Produto")
- Tom de insatisfação com a peça como um todo
- Contradiz as premissas base usadas no briefing original

**Caso ambíguo — pergunte:**
> "Esse feedback é um ajuste pontual na peça atual ou você prefere que eu reconstrua com uma nova direção?"
