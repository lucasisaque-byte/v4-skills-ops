# Storytelling Executivo — Reuniões e Check-ins

## Por que isso importa

Reuniões geram decisões, contexto e próximos passos — mas esse valor se perde quando não há registro estruturado. O Account Manager transforma o material bruto de uma reunião em uma narrativa útil: que avança projetos, alinha expectativas e orienta ação.

---

## Estrutura padrão de storytelling de reunião

```
## [Tipo de reunião] — [Cliente] — [Data]

### Contexto
O que motivou essa reunião. Onde o projeto estava antes dela.

### Objetivo
O que se pretendia resolver, decidir ou alinhar.

### O que avançou
Decisões tomadas. Alinhamentos fechados. Aprovações obtidas.

### O que travou
Pontos em aberto. Divergências não resolvidas. Bloqueadores identificados.

### Leitura analítica
Interpretação do Account Manager sobre o que aconteceu na reunião —
o que o cliente está sinalizando, qual é o padrão de comportamento,
quais são os riscos implícitos.

### Riscos
O que pode comprometer o projeto se não for endereçado.

### Plano de ação
| Ação | Responsável | Prazo |
|------|-------------|-------|
| ... | ... | ... |

### Próximos passos
O que acontece agora. Quem faz o quê. Quando nos reunimos de novo.
```

---

## Tipos de reunião e seus focos

### Check-in semanal / quinzenal
Foco: progresso percebido + alinhamento de prioridades + resolução de bloqueadores
Destaque: o que avançou desde o último check-in, o que está travado

### Kickoff de projeto / campanha
Foco: alinhamento de expectativas + definição de escopo + primeiros passos
Destaque: o que foi acordado, quem é responsável por quê, critérios de sucesso

### Reunião de aprovação
Foco: feedback sobre entregáveis + direção para próxima etapa
Destaque: o que foi aprovado, o que precisa de ajuste, qual é a nova direção

### Reunião de performance / resultado
Foco: análise de dados + diagnóstico + próximos passos estratégicos
Destaque: o que os números dizem, o que vamos mudar, o que vamos testar

### Reunião comercial (cliente novo ou upsell)
Foco: dores do cliente + fit do serviço + próximos passos comerciais
Destaque: o que o cliente valoriza, quais objeções surgiram, qual o próximo passo no funil

---

## Como escrever a leitura analítica

A leitura analítica é onde o Account Manager adiciona valor além do resumo factual.
Ela responde: "O que realmente aconteceu nessa reunião?"

Perguntas úteis para construí-la:
- O que o cliente estava sinalizando nas entrelinhas?
- Qual foi o nível de engajamento e satisfação percebido?
- Há algum padrão de comportamento que se repete?
- O que o cliente não disse, mas estava implícito?
- Qual é o risco real se a equipe não agir nos próximos X dias?
- Há desalinhamento entre o que foi dito e o que foi decidido?

---

## Tom da narrativa

- Direto e executivo — sem floreios, sem rodeios
- Orientado à ação — toda leitura deve ter uma implicação prática
- Honesto sobre riscos — não minimize problemas para "não preocupar o cliente"
- Claro sobre próximos passos — quem faz, o quê, quando

## Template para despacho pós-reunião

Quando a reunião gerar tarefas para outras skills:

```
## Tarefas geradas pela reunião — [Cliente] — [Data]

### Para Copywriting:
- [Descrição da tarefa + contexto da decisão que a gerou]

### Para Social Media:
- [Descrição da tarefa + contexto]

### Para Designer:
- [Descrição da tarefa + contexto]

### Para Account Manager (interno):
- [Acompanhamentos, follow-ups, alertas]
```
