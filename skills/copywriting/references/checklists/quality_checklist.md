# Checklist de Qualidade — 10 Pilares (mínimo 8.5/10)

Avalie cada item de 1 a 10 antes de entregar. Média abaixo de 8.5 = reescreva.

## 1. Clareza (Teste dos 5 Segundos)
- [ ] Um leitor que nunca ouviu falar da empresa entende o que é e para quem é em 5 segundos?
- [ ] A headline não precisa de explicação?
- [ ] Ausência de jargão não explicado?

## 2. Relevância
- [ ] A promessa está diretamente ligada à dor prioritária identificada no UCM/briefing?
- [ ] O nível de consciência do público está correto para o tipo de copy?
- [ ] O cargo/perfil do leitor reconheceria a situação descrita como sua?

## 3. Especificidade
- [ ] Tem pelo menos 1 dado concreto (número, prazo, percentual)?
- [ ] Os benefícios são específicos, não vagos?
- [ ] O mecanismo único é explicado de forma concreta?

## 4. Prova Social
- [ ] Existe pelo menos uma evidência externa (cliente, dado de resultado, certificação)?
- [ ] A prova é do mesmo setor/porte do leitor?
- [ ] Nenhuma prova foi inventada — tudo está no DCC ou briefing fornecido?

## 5. Diferenciação
- [ ] O mecanismo único está claro e é realmente diferente das alternativas?
- [ ] A copy evita clichês de categoria ("solução completa", "end-to-end")?
- [ ] Um concorrente poderia assinar essa copy? Se sim, reescreva.

## 6. Baixa Fricção
- [ ] O CTA pede a menor ação possível para o estágio do funil?
- [ ] Não há parágrafos longos demais (máx. 3-4 linhas)?
- [ ] A copy é escaneável (bullets, hierarquia visual, frases curtas)?

## 7. Fluxo Argumentativo
- [ ] Cada seção leva naturalmente à próxima?
- [ ] A transição entre problema → solução é suave?
- [ ] O leitor nunca precisa "recomeçar" a leitura?

## 8. Força do CTA
- [ ] O CTA diz o que o usuário GANHA, não apenas o que ele FAZ?
- [ ] Há redução de risco associada ("sem compromisso", "em 15 minutos")?
- [ ] O CTA está repetido em pelo menos 2 pontos estratégicos?

## 9. Aderência ao Tom
- [ ] O nível de formalidade está correto para o ICP?
- [ ] Ausência de termos proibidos (ajudar, sinergia, melhor do mercado)?
- [ ] Ausência de palavras robóticas de IA?

## 10. Tratamento de Objeções
- [ ] As 3 principais objeções do UCM/briefing foram abordadas antes do CTA final?
- [ ] As respostas às objeções têm evidências, não apenas promessas?
- [ ] O risco de comprar foi reduzido explicitamente?
