# Identidade Visual — Eduzz

## Estilo geral

- **Estética:** Tecnológica, premium, profissional. Combinação de dados/gráficos com elementos visuais simbólicos e storytelling visual
- **Temperatura:** Paleta fria (azul profundo, branco, cinza) com acentos em amarelo/dourado para destacar CTAs e dados

---

## Carrosséis — padrão visual

**Estrutura por slide:**

| Slide | Elemento visual dominante |
|-------|--------------------------|
| Capa | Imagem forte + headline pesada + dissonância visual |
| Slides de contexto/história | Frames reais (prints, fotos, screenshots do evento) ou imagens geradas por IA |
| Slides de framework | Layout limpo: blocos numerados, ícones lineares, tipografia hierárquica |
| Slides de dados | Números em destaque grande, contexto em fonte menor |
| CTA | Botão "Link na Bio" + logo Eduzz + imagem de CTA do evento/oferta |

**Referência de estrutura narrativa visual:** Notícia → Análise (Frame) → Lição → Convite

---

## Paleta de cores

| Uso | Cor |
|-----|-----|
| Fundo principal | Branco (#FFFFFF) ou azul profundo (#0A1628) |
| Texto principal | Preto (#111111) ou branco (#FFFFFF) conforme contraste |
| Destaque (dados, palavras-chave) | Amarelo/dourado (#FFD700) ou azul Eduzz (#0057FF) |
| Elementos visuais | Gelo/cinza (#E8EDF2) para fundos de slide de framework |
| CTA | Azul Eduzz + branco |

---

## Tipografia

- **Headlines:** Bold, pesada, hierarquia forte — sem serif ou display
- **Corpo:** Regular/Medium, alto contraste com fundo
- **Dados em destaque:** Extra-bold ou Black weight, tamanho grande
- **Máximo por slide:** 3-4 linhas de texto no corpo; headline pode ter 1-2 linhas longas

---

## Estilo visual por tipo de carrossel

### Hot Topics (Newsjacking)
- Slide 1: imagem do evento/personagem real (pinguim, urso Coca-Cola, etc.) + headline com dissonância
- Slides intermediários: frames do evento real + texto de análise
- Transição: do fundo branco/claro (história) → azul escuro (framework) → CTA
- Tom visual: jornalístico com camada de análise estratégica

### Educacional / Framework
- Fundo claro com blocos bem separados
- Ícones lineares (onda, bússola, estrela, setas)
- Layout didático — cada passo do framework ocupa um slide
- Checklist no slide de lição (✅ lista de insights)

### Autoridade / Dados
- Números grandes centralizados
- Fundo escuro com texto branco
- Sub-texto em cinza claro explicando o dado
- Logo Eduzz sempre presente

---

## Vídeo / Reels

- Demonstrações de produto: screen recording da plataforma + voz em off
- Depoimentos: formato entrevista limpo, fundo de escritório ou ambiente neutro
- Educacional curto: slide animado ou apresentação em câmera direta

---

## Logo e marca

- Logo Eduzz: azul escuro (#0057FF) ou branco sobre fundo escuro
- Sempre presente no último slide (CTA) e discretamente no rodapé dos demais
- Não usar o logo em versão muito pequena — mínimo 80px de largura
